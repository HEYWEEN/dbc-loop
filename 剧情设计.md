# 《循环内战 · 地表最菜队》剧情设计文档

> 工作标题暂定,可改。配套阅读:[人物设定.md](./人物设定.md)
> 本文是**叙事圣经(Story Bible)**——定义核心概念、循环规则、五幕结构、角色弧光、系统与结局。

---

## 一、核心概念(一句话)

> **月饼想练技术,被 DBC「地表最菜队」拉去打一局内战,却被困在了这一局的时间循环里——她是唯一记得循环的人,但因为太菜、一紧张就复读,没人信她。**

- **舞台**:**一局 DBC 内战**(封闭场景,固定 5 人 + 小鱼客串)。不是无限排位,是**同一局反复重开**——《土拨鼠之日》《彗星来的那一夜》式密闭循环。
- **配置**:内战 = 4 求生者 vs 1 监管者。月饼**固定求生者**;某坨史/黑檀/薯条/涛儿轮换,其中一人当监管者。
- **DBC** = 「地表最菜队」(拼音首字母梗:**地·表·菜**),队名是自嘲,也是勋章。

## 二、循环规则(机制设定,务必自洽)

| 规则 | 设定 |
|------|------|
| **重置触发** | ① 月饼被绑上**狂欢之椅**淘汰;② 这一局内战结束(无论胜负) |
| **跨循环保留** | **只有月饼的记忆**——她记得发生过的事 + **学到的操作/知识(技能解锁持久化)** |
| **跨循环清空** | 队友的记忆、这一局的进度、地图状态,全部归零 |
| **知情者悖论** | 月饼想揭穿循环,但**结巴 + 复读 + 太菜** → 队友只当她"又犯病了" |
| **UI 锚点** | 屏幕角落的**循环计数器**(致敬《高考模拟器》)+ 速度滑块 + 音效开关 |

> ⭐ **设计要点**:循环保留"记忆与技能"是**教学的合法性来源**——月饼(和玩家)每循环都比上一次强一点,这就是"训练副本"的内核。

## 三、情感内核(全剧的"为什么")— 三层下沉

> 基调:**心理恐怖**。表面温馨搞笑,底下越想越凉。每一层都把上一层重新诠释。

| 层 | 玩家以为 | 其实是 |
|----|---------|--------|
| **表层** | 月饼练技术的训练副本 | —— |
| **第二层**(第三幕初) | 月饼"怕拖累队伍、配不上 DBC"的焦虑具象化 | 这只是循环喂给她的、好理解的"假答案" |
| **★第三层 · 大秘密** | …… | 见 [§三·五]。**这局内战里一直有第五个人,长着月饼的脸。** |

> ⚠️ 反套路通关:玩家会以为"通关 = 赢下内战"。**赢了但没醒 → 冷冰冰坏结局;醒了 → 哪怕输也破循环。**

## 三·五、大秘密(剧透警告 · 全剧地基)

> 心理恐怖路线:**队友都不是真人**——是这场循环(第五人格式的"推演"机制)用月饼的回忆**一遍遍重建的残响**。恐怖朝内、安静、越想越凉。

### 核心真相

- 月饼把这**同一局内战**打了**亿万遍**。久到无法计量。
- 某坨史、黑檀、薯条、涛儿,是残响;残响会**损耗**——越往后,他们能说的新台词越少。
- 那个被无数次循环喂养出来、**精确到帧、强到非人的"真·月饼"**,以监管者之姿藏在地图上,顶着「饼爹」ID,长着她的脸。
- **循环的唯一目的**:让月饼一直相信自己"地表最菜",这样她**永远练不够、永远不走向大门、永远不会醒来面对"队伍早已不在"的事实**。她的菜,是这场循环最精心维护的牢笼。

### 揭示节奏(三连递进,埋在中后段)

| # | 揭示 | 钩子 |
|---|------|------|
| ① | **计数器在骗她** | 终章故障闪过真实循环数——天文数字。呼应"158 自称 160":这世界擅长把数字往好听了报 |
| ② | **队友是残响** | 紧张时世界"卡顿"=残响穿帮;她复读=替演旧的世界补台词 |
| ③ | **怪物长着她的脸** | 监管者互换桥段切视角,瞥见另一个监管者动作和她一模一样,胸前挂「饼爹」 |

### 旧笑点 → 全部重新诠释为线索(心理恐怖的命门)

| 原本的笑点 | 揭开后的真相 |
|-----------|-------------|
| 🔁 **复读**(重复别人最后一句) | 这局她打了亿万遍,每句台词都先于队友知道;复读是回声——她在**替演旧的世界补台词,好让循环演下去** |
| 😰 **一紧张就结巴** | 她慌时**现实跟着卡顿**——循环承压、画面穿帮的瞬间 |
| 🙈 **回头看都难** | 唯一做不到的操作。因为**循环不许她回头**——回头会看见椅子上的真相 |
| 🐷 改名 `江入诗月`→`饼爹`、"不对!不对!" | 身份反复被改写;"不对"是潜意识在抗拒她不敢承认的事实 |
| 👑 **女王蜂"全收集"** | "全收集"=她早已把一切做尽无数遍,**再没有新东西**了 |

### 爽点:菜神觉醒,亲手结束它

- 真·终章反转通关条件——**不是赢,是停止"为变强而留下"**。
- 月饼第一次不为变强而玩,接受自己菜、接受真相。**就在她"认输"的瞬间,被压抑了亿万次循环的真实操作决堤**:玩家操控一个**神乎其技的月饼**——秒密码机、虫海吞掉"它"的视野、从容压点开门。
- 她终于**回了头**,直视椅子上的自己,把队友说过的每一句话**完整、原样、不再因为怕忘而复读地**说一遍,然后推开大门,走出红教堂。
- 「地表最菜队」最菜的那个,在最后一局成了**没人能抓住的神**——不为赢,只为亲手把循环走完。

> **内核一句话**:循环不是要把月饼练成神;是因为她早已是神,而骗她相信自己最菜,是唯一能困住神的牢笼。

## 四、五幕结构(含循环段落)

> 循环数仅为节奏示意,实装可调。

### 序章 · 第一局(Loop 0)— 教学
- 内战开局,大厅 banter 介绍全员,顺手把**基础操作做成教学**(回头看、修机、翻窗——"连回头看都难"就是字面意义的新手教程)。
- 月饼被秒倒、团灭/被淘汰 → 黑屏 → **醒在大厅,计数器跳成 Loop 1**。循环正式启动。

### 第一幕 · 否认(Loop 1~N)— "我是不是疯了"
- 月饼发现一切在重复。试图告诉队友 → **结巴 + 复读 + "不对!不对!"** → 没人信(喜剧高峰)。
- 玩法:**每个循环被动学一个机制**(修机节奏→遛鬼绕板→回头看→救人时机),逐步留存,DBC 慢慢打得没那么烂。

### 第二幕 · 利用(Loop N~M)— "我能改变什么"
- 月饼接受循环,用**预知**主动实验:换打法、试位置、整活(嘴硬"你就说没下来吧"专门气指挥位)。
- 用"多出来的时间"积累**羁绊值**、解锁队友背景(某坨史为何不再指挥、黑檀烈性下的柔软、涛儿的保姆心)。
- 🩸 **恐怖初裂(暗处)**:队友偶尔说出**和上个循环一字不差**的话;月饼一紧张,画面**卡顿一帧**;计数器有时跳得不对。玩家会先于月饼起疑。

### 间奏 · 照面(特殊 Loop)— 切到刀的另一边
- 队友起哄,月饼**第一次当监管者**反追他们,笑果拉满。
- 🩸 **第一次直面"它"**:监管者视角下,地图上**还有另一个监管者**,动作和月饼一模一样、永远慢半步,胸前挂着「饼爹」。月饼以为是 bug,一回头,它没了。**汗毛直立的第一锤。**

### 第三幕 · 假答案 → 真相(Loop X)— "为什么破不了"
- **假答案**:月饼已经能赢,循环却纹丝不动。她以为是"自己还没真正接受自己菜"(第二层焦虑)——按这个去做,**依旧破不了**。
- **真相揭盖**:三连揭示引爆(计数器骗局 / 队友是残响 / 怪物长着她的脸,见 [§三·五])。她终于被允许**回头**,看见椅子上的自己。世界开始大面积穿帮。

### 第四幕 · 觉醒(终循环)— 多结局
- 通关条件反转:**不是赢,是停止"为变强而留下"**。月饼接受真相的那一刻,亿万循环的真实操作决堤——**菜神觉醒**。
- 依据累积的羁绊值与关键选择,导向不同结局(见第七节)。

## 五、角色弧光:每个队友"教会月饼什么"

| 角色 | 教的"技术" | 教的"人生" |
|------|-----------|-----------|
| **某坨史** INFJ | 大局观、救人时机 | "没事没事"的松弛——治她的紧张 |
| **黑檀** INTP | 遛鬼、修机效率 | 烈性:菜就菜,别怂,刚一点 |
| **薯条** ISFP | 手感、牵制节奏 | 随性:凭感觉玩,别想太多 |
| **涛儿** ISFJ | 团队配合、补位 | 包容:他一直在悄悄照顾她情绪 |
| **小鱼** ENFP(客串) | —— | **临门一脚**:玩游戏本来就是为了开心 |

> 四位老玩家 = 月饼的"四种解药";小鱼 = 点破真相的快乐源泉。

## 七、结局树(多结局 = 重玩价值)

| 结局 | 触发 | 基调 |
|------|------|------|
| **真结局 · 菜神觉醒** | 直面真相 + 高羁绊 + 选择"亲手结束" | 菜神觉醒,神乎其技走完最后一局,回头说完每句话、推开大门走出红教堂。**汗毛直立 + 爽**。她带着亿万次循环的温柔,走向真实的当下 |
| **回炉结局 · 又一次"接受自己菜"** | 停在第二层(假答案)、不敢回头 | 她以为自己和解了,循环却悄悄重启——**计数器只多跳了 1**。最细思极恐的坏结局 |
| **共谋结局 · 与"它"合一** | 直面真相但**贪恋强大**、拒绝离开 | 她拥抱"真·月饼"的力量,亲手接管循环,成为新的监管者——下一个困住自己的怪物。爽到发寒 |
| **沉沦结局 · 地表最菜永恒** | 反复逃避真相 | 永远当那个被宠着的菜鸟,永困假世界。温水煮青蛙式的黑色温柔 |
| **彩蛋结局** | 特定整活路线 | "女王蜂全收集" / "你就说没下来吧"荒诞收场,打破第四面墙 |

## 八、笑点 & 名场面清单(必须保留)

> ⚠️ **双重身份**:下列每个笑点,在 [§三·五] 揭秘后都会被重新诠释成**恐怖线索**。写文案时要让它**第一遍好笑、第二遍发凉**——这是本作心理恐怖的命门,别只当段子写。

- 🔁 **复读机**:被追 / 紧张时,语音里复读队友最后一句,不管那句是什么
- 🗣️ **"不对!不对!"**:口头禅,源于改名 `江入诗月` → 自以为"月饼爹"被占 → 一通乱改成"饼爹"后的崩溃
- 😤 **"你就说……没……下来吧!"**:嘴硬,专气(早已不指挥的)指挥位某坨史和好闺蜜黑檀
- 🐷 **雷霆猪**:黑檀所起,因为月饼有时候真的像只猪
- 📏 **158 自称 160**
- 👑 **女王蜂全收集**:冲动抽限定金一路歪到保底的"成就"

## 九、规模估算(对齐"较大体量"目标)

- **循环场景**:序章 1 + 四幕约 N 个关键循环(可扩展到数十个小循环节点)
